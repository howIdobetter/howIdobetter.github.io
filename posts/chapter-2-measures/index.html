2A Outer Measure on R

Motivation and Definition of Outer Measure

The Riemann integral aarises from approximating the area under the graph of a function by using rectangles. To extend integration to a larger class of functions than the Riemann integrable functions, we will write the domain of a functions as the union of subsets more complicated than the subintervals used in Riemann integration. In order to do this, we need a way to measure the size of these subsets. The outer measure is a way to assign a size to any subset of R, not just intervals.

For example, we expect the size of the set (1,3)∪(7,10)(1, 3) \cup (7, 10)(1,3)∪(7,10) to be 555 (because the first interval has length 222, the second interval has length 333, and 2+3=52 + 3 = 52+3=5).
